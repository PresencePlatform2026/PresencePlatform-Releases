PresencePlatform Web Flasher

First-time firmware install for the Waveshare ESP32-S3-ETH board. After this one flash, the device updates itself over the air.

  1. Use Chrome or Edge on a computer (Web Serial doesn't work on phones/iPads).
  2. Connect the board's USB-C port to this computer with a data cable.
  3. Click Connect below and pick the board's serial port (shows as "USB Serial Device" or "USB JTAG/serial debug unit").
  4. Choose Install. For a brand-new board, pick "Erase device" when offered. Takes ~3 minutes.
  5. When it reboots, join the PresencePlatform-Setup WiFi network it broadcasts and follow the setup portal.

Board not showing up? Hold the BOOT button while plugging in USB, then try Connect again. Still nothing — swap the cable; charge-only cables are the #1 cause.

Already-configured device? Choosing "Erase device" wipes its WiFi credentials, admin password, and tracked devices — you'll re-onboard from scratch. Existing devices normally update via Settings → Check for Updates instead of this page.
What gets installed

Bootloader, partition table, OTA data, application, and web dashboard — the same signed artifacts published in the GitHub releases. Future updates are delivered over the air with RSA-2048 signature verification.
